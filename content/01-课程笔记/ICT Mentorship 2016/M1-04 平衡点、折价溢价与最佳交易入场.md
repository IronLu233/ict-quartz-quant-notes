### 从四根蜡烛到回到均衡

![[M1-04_equilibrium_1110.jpg]]

#### 第一步：先用增强 fractal(3) 确认波段高点开始回撤

课程里说的"四根蜡烛"，不是单纯把第 4 根蜡烛也算进一个 fractal，而是先确认一个局部波段高点，再确认回撤已经开始。

- 前 3 根蜡烛，本质上可以理解为一个 `fractal(3)` 式的局部高点
- 中间那根蜡烛是最高点
- 左右两边的蜡烛都低于它
- 回撤确认条件：后续只要有一根K线的收盘价跌破第 3 根K线（高点右侧那根）的最低点，就确认回撤开始
- 不一定是紧挨着的第 4 根，可能是第 5 根、第 6 根才跌破，关键是收盘价跌破第 3 根的最低点
- 反过来，如果是看跌波段的局部低点，后续只要有一根K线的收盘价涨破第 3 根K线的最高点，就确认反弹开始

所以这段逻辑更接近：`fractal(3) + 后续收盘价跌破（或涨破）第3根K线的极值确认回撤开始`。

它的作用不是直接给出买入信号，而是先确认：

- 一段向上的 `impulsive price swing 冲动性价格波段` 已经形成
- 波段高点已经出现

#### 第二步：确认回撤开始后，等待价格回到均衡或折价区

当增强 `fractal(3)` 确认回撤开始以后，ICT 接下来不会马上买入，而是开始等待价格回调。

- 先用这段 `low -> high` 的冲动波段去测量区间
- 重点看这段波段的 `50%`，也就是 [[Equilibrium 均衡]]
- 如果价格回到 `50%`，说明市场回到了公允价值
- 如果价格跌到 `50%` 下方，就进入 `Discount 折价`
- 更理想的是继续回到 `62% - 79%` 的深层折价区，也就是视频里不断强调的 [[OTE 最佳交易入场]]

所以结构上应该理解成：

```text
增强 fractal(3) 确认 -> 确认波段高点与回撤开始 -> 等价格回到 Equilibrium 均衡 -> 再观察是否进入 Discount 折价 / OTE
```

到这一步，仍然还不是直接入场，而是把图表背景建立好。真正的执行，要等价格进入均衡下方以后，再去低时间框架寻找做多理由，例如看涨 [[OrderBlock 订单块]]、旧低点被扫、[[Turtle Soup 海龟汤]]，或者价格是否快速脱离折价区。

### 嵌套 Fib 的两种模式

这节课里出现了两种嵌套使用 `Fib` 的模式，目的不同。

#### 模式一：反弹确认后，用小 Fib 找更好的入场位

![[M1-04_nested_fib_2550.jpg]]

价格先在大波段的 `Discount 折价` 区域产生反应，已经开始向上反弹，甚至回到了溢价区。反弹本身形成了一段新的较小冲动波段，对这段小波段再画 `Fib`，目的是在反弹已经被确认的前提下，找到更好的回调入场位。

- 大波段的折价区已经完成了它的任务：价格从那里反弹了
- 反弹形成的小波段，是一段新的向上冲动
- 对小波段画 `Fib`，等它回调到小波段自己的 [[Equilibrium 均衡]] 或 [[OTE 最佳交易入场]]
- 这时候入场，是在反弹方向已经被确认的基础上，拿到更好的价格

```text
大波段折价区反弹 -> 反弹形成小波段 -> 对小波段画 Fib -> 等小波段回调到 Equilibrium / OTE -> 入场
```

#### 模式二：折价区内部再找折价区，拿更高概率的反弹

![[M1-04_discount_in_discount_3800.jpg]]

价格已经处于大波段的折价区内，还没有反弹。但在折价区内部又出现了一段冲动波段，对这段内部波段再画 `Fib`，就得到了折价区里的折价区。

- 大波段定义了整体的 `Discount 折价` 区域（截图中黄色阴影）
- 价格已经在大波段的 `50%` 下方，处于折价区
- 但在折价区内部，价格又形成了一段新的向上冲动波段
- 对这段内部波段再画 `Fib`，它自身的 [[Equilibrium 均衡]] 和 [[OTE 最佳交易入场]] 又进一步指出了折价区里更深层的买入位置

```text
大波段折价区 -> 内部又出现冲动波段 -> 对内部波段再画 Fib -> 得到折价区里的折价区
```

#### 两种模式的区别

- 模式一：反弹已经发生，方向已被确认，用小 Fib 是为了拿到更好的入场位置
- 模式二：还在折价区里，反弹尚未发生，用小 Fib 是为了找到更高概率会发生反弹的位置

视频里的逻辑是：如果价格已经处于大波段折价区，又进一步跌入了内部小波段的 [[OTE 最佳交易入场]]，那这个位置的看涨概率最高，因为两层折价同时指向同一个区域。

### 不是到了均衡或 OTE 就直接挂单

继续往后看视频可以进一步确认：仅仅因为价格到了某个波段的 [[Equilibrium 均衡]] 或 [[OTE 最佳交易入场]] 就直接挂单买入，本质上是在接飞刀。

实际上，图表上同时存在很多组可能有效的波段，每一组都有自己的均衡和 OTE。价格到了其中一组的位置，不一定真的会反弹。更合理的做法是：

- 先在图表上标出多组可能有效的波段，以及它们各自的均衡和 OTE
- 当价格到达某一组的位置时，不要直接挂单
- 而是切到低时间框架去观察：价格在这里有没有出现反转的迹象
- 如果低时间框架上确实出现了反转信号，那这一组波段就是有效的，可以入场
- 如果没有等到反转，说明这一组波段失效了，不用硬做，继续等下一组位置

### 同一组 Fib 可以反复使用

视频里 ICT 对同一段波段的 `Fib` 反复使用了很多次。只要价格触及均衡或 OTE 后成功反弹、回到了溢价区，这组 `Fib` 就不会失效，下一次价格再回来时仍然可以继续用。

- 一组 `Fib` 的有效寿命不是"用一次就作废"
- 只要底部的波段低点没有被跌破，这组 `Fib` 就还在
- 价格可能多次从溢价区回到均衡或折价区，每次回来时，同一组 `Fib` 的均衡和 OTE 仍然是有效的观察位置
- 真正让一组 `Fib` 失效的条件，是价格跌破了这段波段的起点（低点），说明整个冲动波段被否定了

### 适用于各个时间周期

视频里 ICT 先在日线图上完整演示了整套流程（增强 fractal(3) 确认 -> 等回到均衡 -> 进入折价区 -> 观察反应），随后有人质疑"换到小时图会不会完全不同"，他就直接在小时图上重复了完全一样的逻辑，结果也一样。

- 他明确说：无论你是日内交易者、短期交易者、长线交易者还是波段交易者，都可以运用这一方法
- 他寻找的价格交付方式没有任何改变
- 视频后半段（约 54:31 开始）切到小时图后，也用了同样的步骤：找冲动波段、确认波段高点、等回到均衡、观察折价区反应

所以这套逻辑是跨时间周期通用的，不是只适用于日线。日线、4小时、1小时、15分钟都可以用同样的方式去找冲动波段、确认高点、测量均衡与折价。

- [[Equilibrium 均衡]]

## 要点总结

- 课程里的"四根蜡烛"可理解为增强 `fractal(3)`：先确认局部高点，后续只要收盘价跌破第 3 根K线的最低点就确认回撤开始（不一定是第 4 根）；看跌方向则是收盘价涨破第 3 根K线的最高点
- 确认后的第二步不是立即买入，而是等待价格回到 [[Equilibrium 均衡]]，并进一步观察是否进入 `Discount 折价` 或 [[OTE 最佳交易入场]]
- 价格若已经在较大波段的折价区反弹，就可以继续对反弹形成的较小波段再画一组 `Fib`，用更小的 [[Equilibrium 均衡]] 与 [[OTE 最佳交易入场]] 细化观察
- 嵌套 `Fib` 还有第二种用法：价格仍在大波段折价区内部时，对折价区内出现的冲动波段再画 `Fib`，得到折价区里的折价区，两层折价重叠时看涨概率最高

## 量化上的意义

如果把这节课转成量化研究，重点不一定是人工主观地选出唯一一段 `Fib`，而是先系统地生成一批候选波段，再观察哪些波段的 [[Equilibrium 均衡]] 与 [[OTE 最佳交易入场]] 真正对后续价格产生了反应。

- 可以先用"增强 `fractal(3)`"识别一批候选 swing high / swing low
- 再把这些候选点按方向和时序组合成可能的 `Fib` 起点与终点
- 对每一组候选波段，计算 `50%` 的 [[Equilibrium 均衡]]，以及 `62% - 79%` 的 [[OTE 最佳交易入场]]
- 然后统计价格后续是否在这些位置附近出现回撤停止、明显位移、止损扫荡后反转，或低时间框架上的确认信号

这样做的意义，在于把"手动画 Fib"的主观过程，改成"候选波段集合 + 事后验证"的研究框架。

不过这里不能无约束地把所有高低点两两配对，否则组合会迅速失控。因此更合理的做法是先筛选候选波段，例如：

- 必须先存在明显位移，才能定义为有效冲动波段
- 终点要由"增强 `fractal(3)`"确认
- 波段本身要有最小振幅要求
- 高点确认之后必须真的出现回撤

在这个框架下，[[OTE 最佳交易入场]] 的起点和终点就不再完全依赖主观判断，而是可以由一套可重复的摆点识别规则生成，再由市场后续反应来筛选哪些区间更有效。

此外，嵌套 `Fib` 的用法在量化上也有直接意义：当大波段的 [[Equilibrium 均衡]] 或 [[OTE 最佳交易入场]] 与小波段的 [[Equilibrium 均衡]] 或 [[OTE 最佳交易入场]] 在价格上重合或接近时，说明不同层级的回撤区间指向了同一个位置，这种重叠本身可以作为更高概率的筛选条件。

如果把 [[Equilibrium 均衡]] 和 [[OTE 最佳交易入场]] 的"价格到了这里之后大概率快速脱离"这个特征应用到期权上，也有几种可能的玩法：

- **跨式 / 宽跨式**：当价格正在接近某个 [[Equilibrium 均衡]] 或 [[OTE 最佳交易入场]]，但方向不确定时，可以买跨式或宽跨式，赌波动率爆发
- **垂直价差买方**：当多个不同层级的 [[Equilibrium 均衡]] / [[OTE 最佳交易入场]] 重叠在同一个价格区间，方向偏见更强时，可以用垂直价差买方控制权利金成本
- **裸 Long**：更激进的做法是直接裸买 Call 或 Put，赚 gamma 的钱——价格一旦快速脱离折价区，delta 和 gamma 同时放大，收益会非线性增长

不过裸 Long 的前提是历史数据能支撑胜率。需要先回测：价格触及 [[Equilibrium 均衡]] / [[OTE 最佳交易入场]] 后，在多长时间内发生了多大幅度的位移，以及这个位移能否覆盖掉权利金的时间衰减。如果历史上大部分情况下价格确实快速脱离，那裸 Long 的期望值就是正的；如果经常在折价区磨很久才走，那时间价值会把利润吃掉。

关于如何找到有效的 `Fib`，更程序化的思路是：用 `for` 循环遍历所有"增强 `fractal(3)`"识别出的摆点，将它们按方向和时序两两组合成 `(起点, 终点)` 元组，每个元组对应一组 [[Equilibrium 均衡]] 和 [[OTE 最佳交易入场]]。当价格到达某个元组的均衡或 OTE 位置时，不直接挂单，而是进入低时间框架等待反转信号。如果等到了反转，该元组有效，入场；如果没有等到，标记该元组失效，从候选集合中移除，继续在剩余有效元组的位置等待。这样整个过程就变成了一个"候选元组生成 -> 逐个验证 -> 失效剔除"的循环。

上述"在低时间框架等待反转信号"这一步，可以考虑结合 TTrades 分形模型中的 C2 / C3 蜡烛形态：

- **TTrades C2（反转蜡烛）**：价格扫过前一根蜡烛的高点或低点后，出现反转收盘。影线短则支持后续扩张，影线长则只看回到开盘价附近。
- **TTrades C3（延续蜡烛）**：C2 完成反转后的下一根蜡烛，顺势延续。如果 C2 影线较长不适合直接交易，就等 C3 的延续。

两者和 ICT 框架的结合逻辑是：

- ICT 的 [[Equilibrium 均衡]] / [[OTE 最佳交易入场]] 提供"在哪个价格区域等待反应"的背景
- TTrades 的 C2 / C3 提供"等到什么样的K线形态再确认入场"的触发条件
- 如果在低时间框架上看到 C2 形态出现在 [[OTE 最佳交易入场]] 或 [[Equilibrium 均衡]] 附近（激进入场），或者在高时间框架上已经形成 C2、等到 C3 延续确认（稳健入场），两者叠加可能在量化回测中产生正期望值的 edge
